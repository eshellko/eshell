Command Reference

Command GUI Command Description
exit Exit Exit from eshell tool.
read Read verilog Read verilog file.
clear_edb Clear Design Clear 'eshell.edb' content.
mif2rom Mif2rom Start mif2rom converter.
cpu Sets MAX number of threads. Useful for multi-core processors.
help Start help for eshell.
lp Start Low-Power Advisor for RTL power optimization.
write_verilog Write netlist Write verilog file for defined design.
report_design Report Report designs in memory.
report_port Report Report ports of the current design.
report_object Report Report variables of the current design.
man Help Start manual.
read_vcd Read VCD Read Value Change Dump (VCD) file. Used for Simulation-Aware Power Analysis.
build Builds selected design from modules in 'eshell.edb'.
set_workspace Specifies workspace name
