Инструкция по установке

eshell распространяется в виде двоичного исполняемого файла скомпилированного для 32 и 64-битных версий CPU со статической линковкой библиотек и без статической линковки.

Файл Архитектура Замечания
bin/eshell32.exe 32-бит Linux Для 64-битных CPU может потребоваться модификация переменной LD_LIBRARY_PATH для указания пути к 32-битным библиотекам (ex. export LD_LIBRARY_PATH=$LD_LIBRARY_PATH:/usr/local/lib/)
bin/eshell32s.exe 32-бит Linux со статическими библиотеками -
bin/eshell64.exe 64-бит Linux -
bin/eshell64s.exe 64-бит Linux со статическими библиотеками -


Настройка среды

Дистрибутив содержит скомпилированные программы для нескольких архитектур. Для запуска требуемой архитектуры используется скрипт eshell. В качестве настроек он принимает те же аргументы, что и программа, а также дополнительно позволяет выбрать 32-битную или 64-битную архитектуру.

Если система не содержит требуемых библиотек (например, версия системы отличается от версии, на которой компилировалась программа), то может быть выбрана программа со статической линковкой библиотек. Для ее использования следует определить переменную среды ESHELL_STATIC отличной от пустой строки (например, export ESHELL_STATIC=1).
