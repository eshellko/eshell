Компиляция

Основная работа с дизайном происходит через Verilog описание. Компиляция RTL описания происходит с помощью read. Файлы сохраняются на диск в рабочей директории в <workspace>.edb. Компиляция SystemVerilog packages происходит с помощью read. Файлы сохраняются на диск в рабочей директории в <workspace>.pdb.

В процессе компиляции происходит проверка совместимости RTL с заданным стандартом Verilog (IEEE 1364-2005 для Verilog и IEEE 1800-2017 для SystemVerilog, в зависимости от настройки break_vlog_on_systemverilog).

Очистка базы данных для текущего workspace в рабочей директории выполняется с помощью clear_db.
