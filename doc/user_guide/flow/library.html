Компиляция технологических библиотек

Для своей работы eshell использует базу данных объектов и атрибутов во внутреннем формате. База хранится на диске в директории проекта и имеет имя, соответствующее имени проекта. Очистка базы осуществляется с помощью команды clear_db. При выполнении команды с диска удаляются все относящиеся к текущей базе файлы. Эквивалентно можно удалить файлы с требуемым именем вручную. Ручное удаление может происходить быстрее, поскольку пользователь удаляет имеющиеся файлы, в то время как eshell перебирает все возможные имена файлов с суффиксами от 0 до 99999. Максимальное количество файлов в базе является внутренним ограничением eshell.

Для проведения трансляции дизайна в базис выбранной технологической библиотеки следует скомпилировать ее представление в формате Liberty.

Компиляция библиотек Liberty происходит с помощью read_lib. Файлы сохраняются на диск в рабочей директории в <workspace>.ldb.
