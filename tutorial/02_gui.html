Lab 3. Graphical User Interface (GUI)

In this Lab you will get basic knowledge about GUI features that can be used for small projects management.

You will learn:

  • Eshell Graphical User Interface (EGUI);
  • EGUI project files syntax;
  • Interaction between GUI and console.

    • Activity:

    SETUP

    Before one can use EGUI it should be correctly setup. EGUI is just a set of TCL/TK scripts to simplify usage for those who prefer GUI over console. One of the possible ways to configure EGUI is to set similar to the following lines into ~/.bashrc file. After such setup user can type egui to run EGUI.

    export ESHELL_HOME=/home/lab_user/tools/eshell/
    export PATH=$PATH:$ESHELL_HOME
    alias egui='wish $ESHELL_HOME/tcl/egui.tcl'

    LAB
  • Open console. Move into this tutorial directory (02_gui).
  • Find ethernet.csv file located there. It is project configuration file that cotains list of Verilog sources that can be separated by ';' or by new line. When separated by ';' they are provided to eshell as a single argument for read command.
  • Type egui.



  • EGUI uses .csv files from launch directory to be configured. With default name is eshell.csv. No project is opened, as there is no such file. To select required project point to Workspace button and type project name - "ethernet" (popup lists available options). Press "Enter" or push OK button which will be equivalent to set_workspace ethernet.
